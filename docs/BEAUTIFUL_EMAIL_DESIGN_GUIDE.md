## ğŸ¨ Email Design Best Practices

### 1. **Responsive Design**

```html
<!-- Use max-width instead of width -->
<table style="max-width: 600px; width: 100%;">
  <!-- Content -->
</table>

<!-- Mobile-friendly font sizes -->
<p style="font-size: 16px; line-height: 1.6;">
  Text content
</p>
```

### 2. **Safe Color Palette**

```css
/* Your Madison Studio brand colors */
--primary: #C4A962;      /* Gold */
--dark: #2D2D2D;         /* Dark gray */
--bg: #F5F1E8;           /* Cream */
--text: #666666;         /* Medium gray */
--light-gray: #999999;   /* Light gray */
--border: #E5E5E5;       /* Border gray */
```

### 3. **Typography Hierarchy**

```html
<!-- H1: Main heading -->
<h1 style="font-size: 28px; font-weight: 600; color: #2D2D2D; line-height: 1.3;">
  Main Heading
</h1>

<!-- H2: Section heading -->
<h2 style="font-size: 20px; font-weight: 600; color: #2D2D2D; line-height: 1.4;">
  Section Heading
</h2>

<!-- Body text -->
<p style="font-size: 16px; color: #666666; line-height: 1.6;">
  Body text
</p>

<!-- Small text -->
<p style="font-size: 14px; color: #999999; line-height: 1.5;">
  Small text
</p>
```

### 4. **Buttons (Bulletproof)**

```html
<!-- Primary button -->
<table cellpadding="0" cellspacing="0" border="0">
  <tr>
    <td style="border-radius: 6px; background-color: #C4A962;">
      <a href="{{url}}" style="display: inline-block; padding: 16px 32px; color: #2D2D2D; text-decoration: none; font-weight: 600; font-size: 16px;">
        Click Here â†’
      </a>
    </td>
  </tr>
</table>

<!-- Secondary button -->
<table cellpadding="0" cellspacing="0" border="0">
  <tr>
    <td style="border-radius: 6px; background-color: #2D2D2D;">
      <a href="{{url}}" style="display: inline-block; padding: 14px 28px; color: #FFFFFF; text-decoration: none; font-weight: 600; font-size: 14px;">
        Secondary Action
      </a>
    </td>
  </tr>
</table>
```

### 5. **Spacing System**

```css
/* Consistent spacing */
--space-xs: 8px;
--space-sm: 12px;
--space-md: 16px;
--space-lg: 24px;
--space-xl: 32px;
--space-2xl: 40px;
```
