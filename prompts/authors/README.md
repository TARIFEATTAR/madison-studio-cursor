# Madison Studio Author Profiles

This directory contains training profiles for legendary copywriters that Madison can channel.

## Current Authors

1. **Gary Halbert** (`halbert.md`) - Urgency with substance, direct response
2. **Claude Hopkins** (`hopkins.md`) - Reason-why architecture, process explanation
3. **David Ogilvy** (`ogilvy.md`) - Specificity & proof, research-driven
4. **Eugene Schwartz** (`schwartz.md`) - Awareness strategy, progressive disclosure
5. **J. Peterman** (`peterman.md`) - Narrative storytelling, identity transformation

## File Structure

Each author has two versions:

### Quick Reference (`[author].md`)
- **Format**: Concise ~40-50 line reference guide
- **Purpose**: Quick lookup for Madison during content generation
- **Structure**:
  - When to Channel [Author]
  - Core Principles
  - Signature Techniques
  - Rhythm & Structure
  - Example Transformation
  - Checklist

### Complete Training Document (`[author]_complete.md`)
- **Format**: Comprehensive training document (1000+ lines)
- **Purpose**: Deep dive for advanced training and reference
- **Contains**: 
  - Biographical context
  - Philosophy & worldview
  - 30+ gold excerpts
  - Advanced techniques
  - Do/Don't guides
  - Structure formulas

## Usage

### For Quick Reference
The condensed files (`halbert.md`, `ogilvy.md`, etc.) are used directly in prompts for quick style guidance.

### For Deep Training
The complete files can be:
1. Uploaded via **Settings ‚Üí Madison Training** tab as training documents
2. Referenced for advanced technique learning
3. Used to build comprehensive training datasets

## Adding New Authors

1. Create a condensed version (`[author].md`) following the existing format
2. Optionally create a complete version (`[author]_complete.md`) for deep training
3. Update this README with the new author
